# APOD Explorer – UI/UX Prototype

A UI/UX prototype for an Astronomy Picture of the Day (APOD) explorer app, designed to make NASA’s space imagery more accessible, engaging, and easy to browse.

This project focuses on clean layout, intuitive navigation, and a smooth user experience — created entirely in Figma (screens only, no live app).

👉 High‑fidelity screens (not interactive):

---

## 🧩 Key Features

- Home screen showing the Astronomy Picture of the Day  
- Date picker to explore past APODs  
- Random APOD feature for discovery  
- Detailed APOD view with image, description, and metadata  
- Clean, minimal UI inspired by modern space‑themed apps  

---

## 🧠 Design Process

- Analysed NASA APOD user needs and common browsing patterns  
- Sketched initial layouts and user flows  
- Designed high‑fidelity screens in Figma  
- Focused on accessibility, readability, and visual hierarchy  
- Created a consistent visual style using dark mode + accent colours
